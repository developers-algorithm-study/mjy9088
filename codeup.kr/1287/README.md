# 구구단을 *로 출력하기

[1287](http://codeup.kr/problem.php?id=1287)

시간 제한: 1 Sec

메모리 제한: 128 MB



## 문제 설명

구구단을 입력받아 *로 출력하라.

입력받은 수가 1~9사이의 수인지 확인한다.



## 입력

n단이 입력된다.(1<=n<=9)



## 출력

출력예시를 참고하여 구구단을 *로 출력한다.



## 입력 예시

```
2
```



## 출력 예시

```
**
****
******
********
**********
************
**************
****************
******************
```
