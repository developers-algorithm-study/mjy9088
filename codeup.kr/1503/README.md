# 지그재그 입력(2차원 배열)

[1503](http://codeup.kr/problem.php?id=1503)

시간 제한: 1 Sec

메모리 제한: 128 MB



## 문제 설명

하나의 정수N을 입력받아 다음과 같이 작성하시오.

지그재그로 출력하시오.

N이 5라면 다음과 같이 출력한다.

1 2 3 4 5

10 9 8 7 6

11 12 13 14 15

20 19 18 17 16

21 22 23 24 25



## 입력

한개의 정수가 입력된다. (2 <= N <= 50)



## 출력

N*N배열을 지그재그 형태로 출력한다. (숫자와 숫자사이는 공백으로 구분한다.)



## 입력 예시

```
3
```



## 출력 예시

```
1 2 3 
6 5 4 
7 8 9 
```
