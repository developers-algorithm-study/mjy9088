# 2차원 배열 채우기 4(역달팽이 배열)

[1506](http://codeup.kr/problem.php?id=1506)

시간 제한: 1 Sec

메모리 제한: 128 MB



## 문제 설명

평소 호기심이 많은 남호는 정보고 사이트에 있는 달팽이 배열 문제를 다르게 만들고 싶었다.

입력 예시와 출력 예시를 참고 하여 n을 입력 받아 출력하는 프로그램을 작성하시오.



## 입력

배열의 크기 n이 입력된다. (n은 15보다 작은 홀수)



## 출력

역 달팽이 배열을 출력한다.



## 입력 예시

```
5
```



## 출력 예시

```
1 16 15 14 13 
2 17 24 23 12 
3 18 25 22 11 
4 19 20 21 10 
5 6 7 8 9 
```
