# 복면산 문제 2

[2014](http://codeup.kr/problem.php?id=2014)

시간 제한: 1 Sec

메모리 제한: 128 MB



## 문제 설명

복면산 문제 2탄.



```
  ★ ▲ ■
-    ★ ▲
-----------
     ■ ■
```

위 복면산 문제를 푸는 프로그램을 작성하시오.



## 입력

입력은 없다.



## 출력

★▲■-★▲=■■ 형식으로 출력한다.

여기서 특수문자에 해당되는 숫자를 넣어 출력한다.

답이 여러개일 경우 여러줄에 걸쳐 출력한다.
