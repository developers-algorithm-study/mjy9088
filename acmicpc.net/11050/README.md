# 이항 계수 1

[11050](https://www.acmicpc.net/problem/11050)

시간 제한: 1 초

메모리 제한: 256 MB

## 문제

자연수 $N$과 정수 $K$가 주어졌을 때 이항 계수 $\binom{N}{K}$를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 $N$과 $K$가 주어진다. ($1 \le N \le 10$, $0 \le K \le N$)

## 출력

$\binom{N}{K}$를 출력한다.

## 예제 입력 1

```text
5 2
```

## 예제 출력 1

```text
10
```
